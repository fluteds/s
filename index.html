URL Shortener

Hosted by Github Pages

This is a minimal URL shortener that can be entirely hosted on GitHub pages. It does not need the maintenance of any servers or databases and can be hosted entirely on GitHub for free! This repo is forked from nelsontky/gh-pages-url-shortener and maintained.

Link to Repo: dont-ghost.me/1

Usage

To add a new link:

* Open an issue with the title being the link you want to shorten including the http(s):// to /sdb/issues.
* The newly created short url can be accessed via the users default GitHub pages domain or a custom domain (if enabled.)
* The short url will be automatically generated and the issue will be closed.

Features

Unlike many URL shorteners, this one uses a "database" in the form of GitHub issues and can be entirely hosted via GitHub pages.
